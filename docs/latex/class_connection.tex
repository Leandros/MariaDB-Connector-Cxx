\subsection{Detailed Description}
The main class of the Maria\+D\+B connection. Create a connection to a database with {\ttfamily \hyperlink{class_connection_aaa6f9b14267bb48e1c778c9a0c822d50}{Connect()}}, and disconnect with {\ttfamily \hyperlink{class_connection_ab47840ece7746e9de9eb78d3c9ac2a7a}{Close()}}.

If the \hyperlink{class_connection}{Connection} destructor is called, the connection is closed.
